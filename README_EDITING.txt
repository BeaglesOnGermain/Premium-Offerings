How to edit text:
1. Open the page you want to change in a plain text editor such as Notepad, VS Code, or GitHub's web editor.
2. Search for the exact sentence you want to edit.
3. Change only the text between the HTML tags, not the tag names themselves.
   Example: <p>Change this sentence only.</p>
4. Save, then refresh the page in your browser.
